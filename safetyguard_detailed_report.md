# SafetyGuard (통합 안전관리 시스템) 상세 기능 보고서

## 1. 개요 (Overview)
**SafetyGuard**는 백화점/아울렛 등 복잡한 유통 환경의 공사 현장을 위해 설계된 **지능형 안전관리 플랫폼**입니다. 단순 기록을 넘어 AI가 위험을 실시간으로 감지하고, 복잡한 결재 프로세스를 모바일로 일원화하여 현장의 안전 불감증을 원천 차단합니다.

---

## 2. 기존 방식 vs SafetyGuard 상세 비교

| 항목 | 기존 방식 (Paper/Excel) | SafetyGuard (AI Cloud) |
| :--- | :--- | :--- |
| **점검 기록** | 수기 기록 후 별도 타이핑 (중복 업무) | 현장에서 즉시 모바일 입력 (실시간 동기화) |
| **사진 관리** | 개인 폰 저장 후 메일/카톡 전송 | 클라우드에 위치/시간별 자동 자동 분류 저장 |
| **위험 판단** | 관리자 주관에 따른 판단 (편차 발생) | **Gemini AI 비전** 기반 객관적 위험 판별 |
| **결재 승인** | 대면/서면 결재 (평균 1~2일 소요) | 모바일 전자서명 & 알림톡 (평균 1시간 이내) |
| **데이터 자산** | 보관/검색이 어려운 정적인 데이터 | AI가 트렌드를 분석하는 살아있는 데이터 자산 |
| **사후 조치** | 조치 여부 확인을 위한 재방문 필수 | AI가 조치 결과 사진을 자동 검수하여 승인 |

## 3. 주요 모듈별 상세 기능 (Detailed Features)

### [모듈 1] 일일 현장 안전 점검 (Field Inspection)
현장 담당자가 매일 실시하는 루틴 점검으로, 각 직렬별 맞춤형 인터페이스를 제공합니다.

*   **직렬별 맞춤형 점검 (Role-based UI)**
    *   **시설관리**: 전기/기계 설비 중심의 점검 표출.
    *   **안전관리**: 공사 금일 주요 진행 작업(용접, 도장 등) 입력 의무화.
    *   **영업관리**: 매장 레이아웃 및 고객 동선 안전 중심 점검.
*   **4대 핵심 체크리스트**
    1.  **PPE(보호구)**: 안전모, 안전화 등 작업자 보호구 착용 상태.
    2.  **화재안전**: 소화기 비치, 화기 작업 구역 격리 상태.
    3.  **전기안전**: 분전반 상태, 배선 노출 여부.
    4.  **환경관리**: 자재 정리정돈 및 고객 통행로 확보.
*   **AI 실시간 사진 분석 (Core AI)**
    *   사진 촬영 즉시 AI가 위험 요소(예: "안전모 미착용 포착")를 분석하여 위험도를 `정상/주의/경고`로 자동 제안.
    *   분석 내용을 바탕으로 점검 메모를 자동 초안 작성.
*   **지능형 사후 조치 (Corrective Action)**
    *   `경고` 등급 발생 시 즉시 조치 모드로 전환.
    *   작업자가 조치 사진 업로드 시 AI가 "지적 사항이 해결되었는가"를 검수하여 자율 조치 완결성을 확보.

### [모듈 2] 수시 위험성평가 (Risk Assessment)
법규 준수 및 고위험 공사를 위한 다단계 전자 결재 모듈입니다.

*   **7대 공종별 위험성 평가**
    *   천장, 바닥, 벽체/기둥, 집기, 소방설비, 전기, 기타(표지판 등).
    *   각 항목별 `양호 / 보완 / 해당없음` 3단계 정밀 평가.
*   **증빙 기반 증적 관리 (Evidence)**
    *   위험 요소에 대해 **개선 전(Before) / 개선 후(After)** 사진을 1:1로 매칭하여 등록.
    *   각 사진별 구체적인 위험 요인과 조치 사항 텍스트 기록.
*   **4단계 모바일 전자결재 (Approval Flow)**
    *   `기안자(영업) → 영업팀장 → 지원팀장(협조) → 점장(최종승인)`.
    *   각 결재자는 모바일 터치 패드로 직접 서명 및 의견(Comment) 등록.
*   **정식 공문 형태의 PDF 출력**
    *   결재된 모든 이력과 서명, 사진 별첨이 포함된 고해상도 PDF를 즉시 생성하여 법적 증빙 자료로 활용.

### [모듈 3] 본사 통합 관제 (HQ Dashboard)
전국 지점의 안전 상태를 한눈에 파악하고 전략적으로 대응합니다.

*   **글로벌 안전 KPI 대시보드**
    *   지점별 점검 이행률, 위험 발생 빈도, 미점검 공사 현황 실시간 차트 시각화.
    *   직렬별(시설/안전/영업) 점검 참여도 분석.
*   **AI 전략 인사이트 (Strategic Insights)**
    *   전체 데이터를 AI가 읽고 "전반적으로 양호하나 특정 지점의 고소 작업 위험이 증가하고 있음"과 같은 경영 보고용 핵심 인사이트 자동 생성.
*   **공사 최종 안전 보고서 (Final Report)**
    *   개별 공사 종료 시 AI가 전체 기간의 점검 로그를 전수 조사하여 최종 수료/보완 여부를 문장형으로 요약.

## 4. 시스템 기술 요소 (Technical Highlights)

*   **Intelligence**: Google Gemini (LMM) 기반 비전 인식 및 텍스트 요약.
*   **Cloud Eco**: Firebase Real-time DB를 통한 현장-지칭-본사 간의 0.1초 데이터 동기화.
*   **Communication**: Aligo API 연동을 통한 실시간 카카오 알림톡 전송 (결재 요청 알림).
*   **Mobility**: PWA(Progressive Web App) 기술로 별도의 앱 설치 없이 웹으로 즉시 현장 사용.

---

## 5. 기대 효과 (Future Value)

1.  **전사적 안전 수준 상향 평준화**: AI가 일관된 잣대로 위험을 판단하여 관리 격차 해소.
2.  **보여주기식 점검 탈피**: 사진 증빙과 실시간 서명이 없으면 결재가 불가능한 '실질적 안전 체계'.
3.  **사고 예방 비용 절감**: 중대재해 발생 시의 천문학적 리스크를 AI 기반 선제적 조치로 방어.
4.  **브랜드 가치 제고**: 고객에게 안전한 쇼핑 공간을 제공한다는 신뢰 자산 축적.
